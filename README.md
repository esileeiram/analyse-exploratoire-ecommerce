# analyse-exploratoire-ecommerce
Statistiques descriptives : théorie, interprétation et application sur un dataset e-commerce

## Objectif
Ce notebook explore les fondamentaux des statistiques descriptives à travers un dataset e-commerce de 15 000 transactions. Il mêle définitions, 
interprétations et applications pratiques en Python. Chaque concept est défini, illustré et interprété sur une variable réelle du dataset : montant_panier.

## Contenu
- Mesures de tendance centrale (moyenne, médiane, mode)
- Mesures de position (quartiles, quantiles, percentiles)
- Mesures de dispersion (variance, écart-type, IQR, CV, étendue)
- Détection des outliers
- Interprétation des résultats sur la variable montant_panier
- Forme de la distribution (asymétrie et aplatissement)
- Visualisations (histoghramme, KDE, boxplot) 

## Dataset
15 000 transactions e-commerce avec les variables suivantes : données clients (âge, sexe, revenu), comportement d'achat 
(montant, canal, fréquence) et satisfaction.

## Stack technique
- Python 3.x
- Pandas, NumPy, Matplotlib, Seaborn, Scipy
